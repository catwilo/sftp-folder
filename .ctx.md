# sftp-folder — context
## status: STABLE
Shares a local folder over SFTP with atomic locking. Single command, no permanent server.
Targets: Debian + Termux (non-root).
## entrypoint
- sftp-folder.sh -d <dir> [-p port]
## confirmed
- platform detection (Debian/Termux)
- atomic lock prevents conflicting sessions
